Este site funciona como uma vitrine digital para apresentação de templates, packs para redes sociais e outros produtos digitais da ALFA Web & Social.

As vendas não são processadas diretamente por este site hospedado no GitHub Pages. Ao clicar em “Comprar na Hotmart”, o usuário será redirecionado para a página oficial do produto na Hotmart.

Todos os produtos são digitais e editáveis, permitindo que o comprador altere textos, imagens, cores e demais elementos conforme as instruções disponibilizadas.
